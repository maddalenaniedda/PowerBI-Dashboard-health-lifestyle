# Power BI: Health and Lifestyle Dashboard
<img width="1370" height="763" alt="image" src="https://github.com/user-attachments/assets/ef63f10e-74ad-4979-99a1-52a5c32b866d" />


This dashboard visualizes health and lifestyle data to make insights clear, simple, and engaging.

## Key Features
- Accessible color theme for inclusivity
- Interactive filters and dynamic titles
- Visual symbols to communicate data intuitively
- Data cleaning and transformation using DAX
- Connection to CSV data sources via BigQuery / Big Data tools

## Tools
- Power BI
- DAX (for data transformations)
- BigQuery / Big Data connector (for CSV source integration)

## Data Source
The data used in this dashboard is **synthetic** and was sourced from Kaggle:  
[Health and Lifestyle Dataset](https://www.kaggle.com/datasets/rehan497/health-lifestyle-dataset)

## Philosophy
*"Turning data into stories anyone can understand."*
